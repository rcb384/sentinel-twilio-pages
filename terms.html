Sentinel terms and conditions

Terms for the Sentinel SMS program

These terms apply to users who opt in to receive text messages from Sentinel.

Program description

Sentinel sends user-requested reminders, assistant follow-ups, and related informational messages. Messages are sent only after the user gives explicit consent.

Consent

By opting in, the user authorizes Sentinel to send SMS messages to the mobile number they provided. Consent is not a condition of purchase.

Message frequency and fees

  • Message frequency varies based on the user's activity and requested reminders.
  • Message and data rates may apply.

Opt-out and help

  • Reply STOP at any time to cancel future SMS messages.
  • Reply HELP for assistance.

Carrier notice

Carriers are not liable for delayed or undelivered messages.

Privacy

Information collected through the SMS program is handled according to the Privacy Policy.

Changes to the program

Sentinel may update these terms to reflect changes in the messaging program, legal requirements, or operating practices. The published version on this page controls.
