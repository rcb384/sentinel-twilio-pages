Sentinel messaging program

SMS opt-in and consent information

Sentinel sends conversational updates, reminders, and requested follow-ups only after a user explicitly opts in. This page documents the opt-in flow and required disclosures for SMS campaign review.

Message frequency varies Reply STOP to opt out Reply HELP for help Message and data rates may apply

How users opt in

  1. The user requests Sentinel messaging through a first-party workflow.
  2. The user provides their mobile number directly to the Sentinel operator.
  3. The user gives explicit consent to receive SMS updates from Sentinel.
  4. Sentinel sends only the conversational or reminder messages the user requested.

Opt-out and support

  • Reply STOP at any time to cancel SMS messages.
  • Reply HELP for assistance.
  • Consent is not a condition of purchase.
  • Carriers are not liable for delayed or undelivered messages.

Suggested consent language

This copy can be used in a sign-up form or intake flow where the user enters their phone number.

By providing your mobile number and opting in, you agree to receive SMS messages from Sentinel related to requested reminders, assistant follow-ups, and account-related updates. Message frequency varies. Message and data rates may apply. Reply STOP to cancel and HELP for help. See the Privacy Policy and Terms and Conditions.
